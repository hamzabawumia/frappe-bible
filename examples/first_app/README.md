# Example: first_app (scaffold instructions)

This folder contains a minimal example scaffold and instructions to produce a runnable app used in the book tutorials.

How to generate the scaffold locally
1. In your bench directory (see docs for bench init):
   bench new-app first_app
2. Install on a site:
   bench --site site1.local install-app first_app
3. Follow docs/03-first-app/tutorial.md to add DocTypes and code.

Docker quick-run (recommended for labs)
This repository includes lab instructions that use a prebuilt Docker Compose stack. See exercises/lab-01-setup.md for details.
